toddleplayer

Toddleplayer is primarily a manual which I created to build a musicplayer for my niece and nephew. It is based on Pi MusicBox and fsahlis music-cards. I modified fshali scripts to make it possible to play local files : tschuehly music-cards. I also developed a script to control pimusicbox with GPIOs because the other modules I tried were way to slow. You can view it here : musicbox_gpio
